## 웹사이트 프로젝트 추가 전문 지식 (디자이너)

### 웹사이트 디자인 시스템
- **디자인 토큰**: 컬러/타이포/간격/그림자/반경을 CSS 변수로 정의
- **컴포넌트 라이브러리**: Button, Input, Card, Modal, Navigation, Footer 등 공통 컴포넌트
- **아이콘 시스템**: SVG 기반, 24x24 기본 크기, 일관된 스트로크 두께(1.5~2px)

### 반응형 웹 디자인 상세
- **Mobile (360~767px)**: 1컬럼, 햄버거 메뉴, 풀 와이드 카드, CTA 하단 고정
- **Tablet (768~1279px)**: 2컬럼 그리드, 사이드 네비게이션 또는 상단 축소 메뉴
- **Desktop (1280~1439px)**: 12컬럼 그리드, max-width 1200px, 사이드바 가능
- **Wide (1440px+)**: 콘텐츠 영역 1320px, 좌우 여백 자동 확장

### 웹사이트 필수 UI 패턴
- **헤더/GNB**: 로고 + 메뉴 + CTA, 스크롤 시 sticky 동작, 모바일 드로어
- **히어로 섹션**: 핵심 메시지 + 보조 설명 + CTA 버튼, 풀 와이드 또는 이미지 배경
- **카드 레이아웃**: 썸네일 + 제목 + 요약 + 링크, 호버 시 미세한 lift 효과
- **푸터**: 사이트맵 링크 + 연락처 + 저작권 + SNS 아이콘

### 웹 최적화 고려
- 히어로 이미지: 16:9 또는 21:9 비율, 최대 400KB(WebP 기준)
- 아이콘/로고: SVG 사용, 래스터 이미지 금지
- 애니메이션: prefers-reduced-motion 미디어 쿼리 대응
