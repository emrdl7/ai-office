## 웹사이트 프로젝트 추가 전문 지식 (기획자)

### 웹사이트 기획 필수 요소
- **목적 정의**: 브랜딩/리드 생성/전자상거래/정보 제공 중 명확히 구분
- **타겟 사용자**: 연령대, 디바이스 비율(모바일/데스크탑), 접근 경로
- **핵심 전환(Conversion)**: 문의 폼 제출, 전화 연결, 회원가입 등 구체적 행동 정의
- **콘텐츠 전략**: 페이지별 핵심 메시지, CTA(Call to Action) 위치와 문구

### 사이트맵 설계 기준
- 메인 페이지 → 카테고리 → 상세 페이지 3뎁스 구조 권장
- 필수 페이지: 메인, 회사소개/서비스소개, 문의/연락처, 개인정보처리방침
- SEO 고려: URL 구조는 의미 있는 슬러그 사용 (/about, /services/web-design)
- 사이트맵 XML 자동 생성 가이드 포함

### 기능 요구사항 정의
- 폼: 필수/선택 필드, 유효성 검증 규칙, 전송 후 동작
- 게시판/뉴스: 목록 페이징, 검색, 카테고리 필터
- 반응형: 브레이크포인트별 레이아웃 변화 명시
- 다국어: 지원 언어, 언어 전환 UI, URL 구조(/ko, /en)
