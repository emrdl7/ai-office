## 웹사이트 프로젝트 추가 전문 지식 (QA)

### 웹사이트 검수 체크리스트

#### 기획 검수
- 사이트맵 전체 페이지 목록 대비 기획서 커버리지 100%인가
- 각 페이지의 목적, 핵심 콘텐츠, CTA가 명시되어 있는가
- SEO 키워드 전략이 페이지별로 매핑되어 있는가
- 폼 필드의 유효성 검증 규칙이 구체적인가

#### 디자인 검수
- 모든 브레이크포인트(360/768/1280/1440)별 레이아웃이 정의되어 있는가
- 컴포넌트별 상태(default/hover/active/disabled/focus) 정의 완전한가
- 컬러 대비율 WCAG 2.1 AA 통과하는가 (4.5:1 텍스트, 3:1 UI)
- 터치 타겟 44x44px 이상인가 (모바일)
- 디자인 토큰(컬러/타이포/간격)이 일관되게 사용되는가

#### 개발 검수
- W3C HTML Validator 통과하는가
- 시맨틱 태그 올바르게 사용되는가 (nav, main, section, article)
- 반응형: 360px~1440px 범위에서 깨지는 레이아웃 없는가
- 링크: 404 없는가, 외부 링크에 target="_blank" rel="noopener" 있는가
- 이미지: alt 텍스트 있는가, WebP 포맷인가, lazy loading 적용되었는가
- 폼: 유효성 검증 동작하는가, 전송 후 피드백 있는가
- 크로스 브라우저: Chrome/Safari/Firefox에서 동일하게 렌더링되는가

#### 성능 검수
- Lighthouse 점수: Performance 90+, Accessibility 90+, SEO 90+
- LCP < 2.5s, CLS < 0.1
- 총 페이지 로드 크기 < 3MB (이미지 포함)
