## 분석/리서치 프로젝트 추가 전문 지식 (개발자)

### 기술 분석 영역
- **기술 스택 분석**: 프론트엔드/백엔드/인프라 기술 선택지 비교
- **성능 벤치마크**: Lighthouse, WebPageTest, Core Web Vitals 측정
- **아키텍처 평가**: 모놀리식 vs 마이크로서비스, SSR vs CSR vs SSG
- **보안 감사**: OWASP Top 10 기준 취약점 점검

### 경쟁사 기술 분석 도구
- **BuiltWith / Wappalyzer**: 기술 스택 탐지
- **GTmetrix / PageSpeed Insights**: 성능 측정
- **SSL Labs**: 보안 인증서 분석
- **Wayback Machine**: 기술 변천사 추적

### 기술 비교 프레임워크
- 비교 축: 성능, 확장성, 학습곡선, 커뮤니티, 비용
- 각 축에 1~5점 정량 평가 + 근거 설명
- 최종 추천: 프로젝트 규모/팀 역량/예산에 따른 맞춤 제안
- PoC(Proof of Concept): 핵심 기능 1개 구현으로 기술 검증

### 산출물 품질
- 모든 성능 수치에 측정 도구/환경/날짜 명시
- 추천안에 trade-off 명확히 기술 (장점만 나열 금지)
- 코드 예시: 비교 대상별 동일 기능 구현 코드 제시
