    id: str
    type: MessageType    # 'task_request' | 'task_result' | 'status_update'

    `server/orchestration/router.py` 생성:

    **MessageRouter 클래스:**
    ```python
    class MessageRouter:
        def __init__(self, bus: MessageBus, event_bus: EventBus):
            self.bus = bus
            self.event_bus = event_bus
    ```

    **route(msg: AgentMessage) -> None (async):**
    1. 원본 메시지 bus.publish(msg) 발행
    2. 이미 to_agent == 'planner' 또는 to_agent == 'broadcast' 이면 복사 생략
    3. 그 외: msg.model_copy(update={'id': str(uuid.uuid4()), 'to_agent': 'planner', 'metadata': {**msg.metadata, 'is_broadcast_copy': True}}) 생성 후 bus.publish(copy)
    4. event_bus.publish(LogEvent(agent_id=msg.from_agent, event_type='log', message=f'{msg.from_agent} → {msg.to_agent}: {msg.type}')) 발행

    **주의사항:**
    - AgentMessage는 Field(alias='to')와 Field(alias='from') 사용 → model_copy 시 by_alias=False 기준 to_agent/from_agent 키 사용
    - model_copy(update=...)는 Pydantic v2 방식 — update dict의 키는 field name(alias 아닌 Python 이름)

    import: from bus.message_bus import MessageBus; from bus.schemas import AgentMessage; from log_bus.event_bus import EventBus, LogEvent; import uuid
